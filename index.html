PROGETTI

StoragePlus

Software scritto interamente in Python per una Gestione Magazzino basilare per privati.

Include un database SQL ed il controllo totale sui dati.

MixHUB (Plugin)

Un semplice plugin per Server Minecraft compatibile con Bungecoord/Waterfall che aggiunge il comando /hub personalizzabile.

Supporta il Cooldown e linguaggio personalizzato.

UltimateBackups (Plugin)

Plugin dedicato a Server Minecraft per l'esecuzione di backup automatici o programmati delle mappe caricate.

Utilizza la compressione 1024x in ZIP per il risparmio dati su disco

Pienamente configurabile su autonomia dei backups, orari e linguaggio.
